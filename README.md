# react-native-admob-appopen

React Native AppOpen Format

## Installation

```sh
npm install react-native-admob-appopen
```

## Usage

```js
import AdmobAppopen from "react-native-admob-appopen";

// ...

const result = await AdmobAppopen.multiply(3, 7);
```
